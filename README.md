
# @your-scope/nest-file-manager

Pluggable file storage for NestJS with Local, S3, and GCS engines. Includes an optional REST resource.

## Install

```bash
npm i @your-scope/nest-file-manager
# add engines you need
npm i -S @aws-sdk/client-s3 @aws-sdk/s3-request-presigner
npm i -S @google-cloud/storage
```

See source for usage examples.
